# 2-Weekly-Mission

This project was bootstrapped with [Vite](https://github.com/vitejs/vite) & [Bun](https://github.com/oven-sh/bun)

## Install

Move to root dir and run `bun install`

## Available Scripts

In the project directory, you can run:

### `bun run dev`

### `bun run build`

### `bun run preview`
